# Keisha Esta's To-Do List
This is Homework 3 in the Multimedia Authoring 1 class. I am building a styled and interactive To-Do List web page using HTML and CSS. Starting from the in-class template, the design has been customized with generic selectors, IDs, and classes to create a unique look and feel.

## Installation
No installation required. Simply download the zip file and extract.
## Usage
Open index.html in the browser of your choice.
